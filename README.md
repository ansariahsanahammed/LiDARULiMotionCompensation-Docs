# ULi Motion Compensation — Technical Documentation

This repository contains the complete **LaTeX source code** for the technical documentation only.

>The Python scripts for motion compensation and boresight calibration of point clouds acquired with the **ULi (Underwater LiDAR)** system are maintained separately:
> **[LiDARULiMotionCompensation](https://github.com/ansariahsanahammed/LiDARULiMotionCompensation)**

This work was developed as part of the Master's Thesis:  
**"Development and Analysis of Motion Compensation Algorithms for the Underwater LiDAR ULi"**  
*M.Sc. in Geodesy and Geoinformatics with specialization in Hydrography - HafenCity Universität Hamburg.*
