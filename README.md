# 📘 ULi Motion Compensation — Technical Documentation

This repository contains the complete **LaTeX source code** for the technical documentation of the Python scripts developed for the motion compensation and boresight calibration of point clouds acquired with the **ULi (Underwater LiDAR)** system. 

This work was developed as part of the Master's Thesis:  
**"Development and Analysis of Motion Compensation Algorithms for the Underwater LiDAR ULi"**  
*M.Sc. in Geodesy and Geoinformatics (Hydrography specialization) — HafenCity Universität Hamburg.*
