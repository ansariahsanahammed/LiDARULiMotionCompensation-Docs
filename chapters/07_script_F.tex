\chapter{F\_pulsalyzer\_real\_time\_visualizer.py}
\label{ch:scriptF}

This script sends rotation commands to Pulsalyzer using the \texttt{A\_json\_commander.py} script. This script can be used to visualize the effect of rotation on the Pulsalyzer-rendered point cloud in real-time.

\section{How It Works}
Pulsalyzer accepts a scanner transformation command (\texttt{set\_scanner\_transformation\_deg}) via its JSON WebServer. This script sends commands to Pulsalyzer using the \texttt{A\_json\_commander.py} script in a loop, incrementally updating the scanner transformation angles and triggering a reprocess/redraw of the point cloud in Pulsalyzer.

\section{User-Configurable Parameters}
\begin{table}[!ht]
    \centering
    \begin{tabular}{@{}lll@{}}
        \toprule
        Variable               & Default & Description                                          \\ \midrule
        routing1               & [...]   & Must be updated with actual hostname from Pulsalyzer \\
        x\_rot, y\_rot, z\_rot & 0, 0, 0 & Initial rotation angles (degrees)                    \\
        start, stop            & 0, 10   & Range of angle increments to iterate through         \\ \bottomrule
    \end{tabular}
\end{table}

\section{Prerequisites}
\begin{itemize}
    \item Pulsalyzer must be running with a .lidar file loaded and its JSON server active.
    \item \texttt{A-json\_commander.py} must be in the same directory.
    \item The hostname must match the device name visible in the Pulsalyzer WebServer interface.
\end{itemize}

\section{Running the Script}
\begin{lstlisting}[language=bash]
python F-pulsalyzer_real_time_visualizer.py
\end{lstlisting}
The script applies the initial rotation angles, pauses for a keypress, then enters an infinite loop cycling through the angle range. Press \texttt{Ctrl+C} to stop.
