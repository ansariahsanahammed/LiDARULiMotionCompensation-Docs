\chapter{E\_boresight\_calibration.py}\
\label{ch:scriptE}

This script is an interactive GUI application for visually determining the boresight rotation angles. It uses Open3D for 3D visualization and Tkinter for the control panel. A binary (.exe) file of this script is available under releases in the GitHub repository.

\section{How It Works}
The boresight calibration process involves adjusting three rotation angles (Roll/X, Pitch/Y, Yaw/Z) until the point cloud appears geometrically correct. The tool applies the rigid body transformation in real time as sliders are moved. The angles can also be adjusted as text inputs near the sliders in the control panel.

\section{Usage}

This application opens two windows:
\begin{enumerate}
    \item \textbf{Boresight Calibration Controls} (Tkinter) - Main control panel.
    \item \textbf{Boresight Calibration Viewer} (Open3D) - 3D visualization window.
\end{enumerate}

\subsection*{Boresight Calibration Workflow}
\begin{enumerate}
    \item Click "Point Cloud (.csv)" $\rightarrow$ select the downsampled file obtained from Script C (\autoref{ch:scriptC}).
    \item Click "Trajectory (.txt)" $\rightarrow$ select the Pulsalyzer format trajectory file obtained from Script B (\autoref{ch:scriptB}).
    \item Click "Apply \& View" $\rightarrow$ loads files and shows initial point cloud.
    \item Use the following mouse controls to navigate the 3D viewer:
          \begin{itemize}
              \item \textbf{Left mouse button + drag} --- rotate the view
              \item \textbf{Middle mouse button + drag (or Ctrl + left click and drag)} --- pan the view
              \item \textbf{Scroll wheel} --- zoom in/out
          \end{itemize}
    \item "Reset Camera View" button can be used to reset the camera view and centre the point cloud.
    \item Adjust Roll (X), Pitch (Y), and Yaw (Z) sliders (Text inputs can also be used to set the values).
    \item Note final values for use in the Python workflow (\autoref{ch:pythonWorkflow}).
\end{enumerate}
